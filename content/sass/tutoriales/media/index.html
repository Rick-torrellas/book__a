Media Queries

En sass puedes colocar las media queries dentro del selector del elemento que vas a modificar.

.nav {
    color: blue;
    @media (min-width: 900px) {
        color: purple;
    }
    &__ola {
        width: 100px;
        height: 100px;
        background-color: aquamarine;
    }
}
