## compilar a css

```json
sass ./sass/style.scss ./css/style.css
```

## complitar a css comprimido

```json
sass -s compressed ./sass/style.scss ./css/style.css
```
## autogenerar archivo cuando ocurre un cambio

```json
sass -s compressed -w ./sass/style.scss ./css/style.css
```
